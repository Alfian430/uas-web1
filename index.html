Selamat Datang

Selamat datang di sistem informasi perpustakaan digital. Silakan ikuti panduan berikut untuk menggunakan aplikasi:

  1. Lakukan Login pada menu Login (Gunakan sembarang username).
  2. Masuk ke menu Transaksi untuk mencatat peminjaman buku.
  3. Isi formulir dengan lengkap (Judul, Nama, Kategori, Tanggal).
  4. Klik Simpan, data akan muncul pada tabel di bawahnya.
  5. Cek menu Profil untuk melihat informasi pengembang.
